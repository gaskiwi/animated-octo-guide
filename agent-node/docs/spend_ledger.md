# Spend Ledger

All program spend is recorded here **before** it happens. All payments are human-executed (Yoo). Agents append entries via `guardrails.ledger_append()`; weekly reconciliation by the verifier role.

Budgets: $10,000 hardware hard cap · cloud GPU ≤ $400 cumulative, ≤ $50/run · LLM API $150–250/mo soft cap, $300 hard cutoff (Anthropic console).
